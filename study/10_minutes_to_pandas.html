# Criar dataframe usando dicionario 
'''
Cada chave no dicionrio representa o nome de uma coluna do DataFrame e os valores correspondentes representam os dados para aquela coluna.
As colunas do DataFrame df2 so as seguintes:
A: Uma coluna com o valor constante 1.0.
B: Uma coluna com um objeto Timestamp representando a data "20130102".
C: Uma coluna que contm uma srie com valores 1.0, indexados de 0 a 3.
D: Uma coluna que contm um array NumPy com quatro elementos, todos com o valor 3, com tipo de dados "int32".
E: Uma coluna que contm dados categricos (categorias) representados por objetos Categorical. Os valores so ["test", "train", "test", "train"].
F: Uma coluna com o valor constante "foo".
'''
df2 = pd.DataFrame(
    {
        "A" : 1.0,
        "B" : pd.Timestamp("20130102"),
        "C" : pd.Series(1,index= list(range(4)), dtype = "float32"),
        "D" : np.array([3]*4, dtype = "int32"),
        "E" : pd.Categorical(["test", "train", "test", "train"]),
        "F" : "foo"
    }
)

print(df2)

#Operating with objects that have different dimensionality and need alignment. 
#In addition, pandas automatically broadcasts along the specified dimension:
'''
cria uma nova Series chamada s usando o construtor pd.Series(). 
Essa Series  construda a partir de uma lista de valores [1, 3, 5, np.nan, 6, 8] e um ndice definido como dates. 
Em seguida, o mtodo shift(2)  aplicado na Series resultante.
O mtodo shift()  usado para deslocar (mover) os valores de uma Series para cima ou para baixo ao longo do ndice. 
Nesse caso, shift(2) desloca os valores da Series duas posies para baixo ao longo do ndice. 
Isso significa que os dois primeiros valores da Series so preenchidos com NaN (valores ausentes), 
e os demais valores so movidos para baixo duas posies.
'''
s = pd.Series([1, 3, 5, np.nan, 6, 8], index=dates).shift(2)
print(s)

'''
A funo sub()  usada para subtrair um objeto de outro, elemento a elemento. 
No caso desse cdigo, o DataFrame df  subtrado da Series s, considerando o alinhamento dos ndices.
A operao de subtrao  realizada para cada elemento correspondente entre o DataFrame e a Series. .
Se no houver correspondncia entre os ndices, o valor resultante ser NaN.

HDF5

Um arquivo HDF5 (Hierarchical Data Format 5) um formato de arquivo de dados binrios usado para armazenar grandes volumes de dados de forma eficiente. O HDF5 um formato flexvel, porttil e extensvel que permite organizar, armazenar e gerenciar uma variedade de tipos de dados, incluindo conjuntos de dados multidimensionais, estruturas hierrquicas e metadados.

O formato HDF5 amplamente utilizado em reas que exigem armazenamento eficiente de dados cientficos, como fsica, astronomia, bioinformtica, finanas e cincia dos materiais. Ele capaz de lidar com grandes conjuntos de dados e oferece suporte a recursos como compresso, filtragem, armazenamento de metadados detalhados e acesso eficiente a dados atravs de ndices.

Um arquivo HDF5 pode conter vrios conjuntos de dados, chamados de grupos, organizados em uma estrutura de rvore hierrquica semelhante a um sistema de arquivos. Cada grupo pode conter datasets (conjuntos de dados) que armazenam os prprios dados. Os datasets podem ser multidimensionais, como arrays, ou unidimensionais, como sries temporais.

O formato HDF5 tambm suporta atributos, que so metadados associados a grupos e datasets. Os atributos podem conter informaes adicionais sobre os dados, como unidades, descries, autores, entre outros.

Os arquivos HDF5 podem ser acessados e manipulados usando bibliotecas em vrias linguagens de programao, incluindo Python, MATLAB, Java e C/C++. No contexto do pandas em Python, a biblioteca pytables usada para trabalhar com arquivos HDF5
